# AcademyOS Version 1.0 Launch Program

AcademyOS is feature complete. This folder indexes launch artifacts generated by the **Certification Center** — not a separate operational module.

## Launch reports

- **Platform Health Report:** generated during full certification (Launch Readiness dashboard)
- **Launch Readiness Report:** `docs/launch/LAUNCH_READINESS_REPORT.md` (static) + live copy in Documentation Center after certification

## Launch dashboards

- **Launch Readiness:** `/dashboard/certification/launch`
- **Certification Overview:** `/dashboard/certification/overview`
- **Go Live Center:** `/dashboard/admin/go-live`
- **Demo Organization:** `/dashboard/certification/demo`

## Generated documentation

Run **Regenerate documentation** on `/dashboard/certification/documentation` or **Run full certification** on the Launch Readiness dashboard. Guides include:

Administrator, Teacher, Parent, Finance, HR, Admissions, Executive, Developer, API, Implementation, Student, Support, and Change Log.

## Production checklist

1. Apply migrations through `129_release_v1_doc_categories.sql`
2. Set `CRON_SECRET` in production for `/api/platform/process-queues`
3. Run full certification and review Platform Health Report
4. Resolve Critical and High audit findings
5. Generate demo environment for sales
6. Complete AcademyOS University training paths
